## Summary

Addresses all round-06 blockers + round-07 M-001. Key changes: (1) `read_register` made descriptor-aware via `find_desc()` + `read_with_desc()` for correct shadow CSR semantics; (2) continue loop runs difftest on halting instruction before breaking; (3) GPR names cached in `QemuBackend` from init context, `diff_contexts()` validates length match; (4) `CoreContext` dispatch explicitly `cfg(riscv)` gated in API; (5) xdb state wiring sketch for `respond()`/`mainloop`; (6) `csr_table!` macro extended with `difftest(mask)` annotation — difftest CSR list auto-generated from the table itself, no separate const. CoreContext CSR entries are `(u16, &'static str, u64, u64)` tuples (addr, name, mask, raw) — no `CsrSnapshot` struct.

## Log

[**Review Adjustments**]

- R-001 (read_register shadow semantics): `read_register` now uses `find_desc(addr) -> read_with_desc(desc)` for correct view/shadow behavior on sstatus/sie/sip.
- R-002 (continue drops final instruction): Loop now calls `check_step` before break on halt. Halted flag passed through.
- R-003 (GPR name contract incomplete): `QemuBackend` stores `gpr_names: Vec<&'static str>` from init context. `diff_contexts()` asserts `dut.gprs.len() == ref.gprs.len()`.
- R-004 (non-RISC-V gating): `context()` gated with `#[cfg(riscv)]` on the impl block. `CoreContext` type alias only defined under `#[cfg(riscv)]`. LoongArch gets a compile error if difftest enabled — documented, acceptable for this round.
- R-005 (xdb state wiring): Full `respond()` / `xdb_mainloop()` wiring sketch added.
- R-006 (M-002 claim overstated): Rephrased: whitelist uses `CsrAddr` variants, but context still carries snapshot tuples.

[**Master Compliance**]

- M-001 (eliminate DIFF_CSR_SET): No separate hand-maintained const. Extended `csr_table!` macro with optional `difftest(mask)` annotation per CSR. The macro auto-generates `pub const DIFFTEST_CSRS: &[(CsrAddr, u64)]` from annotated entries. Adding/removing a CSR from difftest = one-line edit in the CSR table.
- M-002 (clean, functional): Iterator chains throughout. Minimal types. No redundant structures.

**`xcore/src/cpu/riscv/csr.rs`** — extend `csr_table!` macro with `difftest(mask)` annotation:

The existing macro syntax `$name = $addr => [$spec]` is extended to accept optional trailing `difftest($mask)`. The macro generates a `DIFFTEST_CSRS` const from annotated entries:

```rust
macro_rules! csr_table {
    ( $( $name:ident = $addr:expr => [ $($spec:tt)* ] $(, difftest($mask:expr) )? ),* $(,)? ) => {

        // ── CsrAddr enum (unchanged) ──
        #[derive(Clone, Copy, Debug, PartialEq, Eq)]
        #[repr(u16)]
        #[allow(non_camel_case_types, dead_code)]
        pub enum CsrAddr { $( $name = $addr ),* }

        // ── find_desc (unchanged) ──
        pub(in crate::cpu::riscv) fn find_desc(addr: u16) -> Option<CsrDesc> {
            match addr {
                $( $addr => Some(csr_table!(@desc $addr, $($spec)*)), )*
                _ => None,
            }
        }

        impl CsrAddr {
            pub fn from_name(name: &str) -> Option<Self> {
                match name { $( stringify!($name) => Some(Self::$name), )* _ => None }
            }
            pub fn name(self) -> &'static str {
                match self { $( Self::$name => stringify!($name), )* }
            }
        }

        // ── Difftest CSR list (auto-generated from annotations) ──
        pub const DIFFTEST_CSRS: &[(CsrAddr, u64)] = &[
            $( $( (CsrAddr::$name, $mask), )? )*
        ];
    };
    // ... existing @desc arms unchanged ...
}
```

The key: `$( $( (CsrAddr::$name, $mask), )? )*` — the inner `$(...)?` only emits for entries with `difftest(mask)`.

**`xcore/src/cpu/riscv/debug.rs`** — context() uses generated `DIFFTEST_CSRS` + fixed read_register:

```rust
use super::context::RVCoreContext;
use super::csr::{CsrAddr, DIFFTEST_CSRS, find_desc};

impl DebugOps for RVCore {
    fn context(&self) -> crate::cpu::CoreContext {
        let gprs = (0u8..32)
            .map(|i| {
                let r = RVReg::from_u8(i).unwrap();
                (r.name(), word_to_u64(self.gpr[i as usize]))
            })
            .collect();

        let csrs = DIFFTEST_CSRS.iter()
            .map(|&(addr, mask)| {
                (addr as u16, addr.name(), mask, word_to_u64(self.csr.get(addr)))
            })
            .collect();

        RVCoreContext {
            pc: self.pc.as_usize() as u64,
            gprs,
            privilege: self.privilege as u64,
            csrs,
            word_size: std::mem::size_of::<crate::config::Word>(),
            isa: if cfg!(isa64) { "rv64imac" } else { "rv32imac" },
        }
    }

    // Descriptor-aware read: correct shadow/view semantics for sstatus/sie/sip
    fn read_register(&self, name: &str) -> Option<u64> {
        match name {
            "pc" => Some(self.pc.as_usize() as u64),
            "privilege" => Some(self.privilege as u64),
            _ => {
                // GPR lookup
                RVReg::from_name(name)
                    .map(|r| word_to_u64(self.gpr[r as usize]))
                    .or_else(|| {
                        // CSR lookup — descriptor-aware for correct view/shadow
                        CsrAddr::from_name(name).and_then(|a| {
                            find_desc(a as u16)
                                .map(|desc| word_to_u64(self.csr.read_with_desc(desc)))
                        })
                    })
            }
        }
    }

    // read_memory, fetch_inst, disasm_raw, breakpoints — unchanged
    // ...
}
```

```rust
fn csr_addr_to_qemu_regnum(addr: u16) -> usize { 4096 + addr as usize }
const QEMU_PRIV_REGNUM: usize = 4161;

pub struct QemuBackend {
    proc: Child,
    gdb: GdbClient,
    gpr_names: Vec<&'static str>,              // from init context
    csr_meta: Vec<(u16, &'static str, u64)>,   // (addr, name, mask) from init context
    word_size: usize,
    isa: &'static str,
}

impl QemuBackend {
    pub fn new(binary_path: &str, reset_vec: usize, init_ctx: &CoreContext)
        -> Result<Self, String> {
        let qemu_bin = if init_ctx.isa.starts_with("rv64") {
            "qemu-system-riscv64"
        } else {
            "qemu-system-riscv32"
        };
        // ... spawn, connect, sstep=0x1, PhyMemMode:1, run to reset_vec ...
        let gpr_names = init_ctx.gprs.iter().map(|(n, _)| *n).collect();
        let csr_meta = init_ctx.csrs.iter()
            .map(|&(addr, name, mask, _)| (addr, name, mask))
            .collect();
        Ok(Self { proc, gdb, gpr_names, csr_meta,
                  word_size: init_ctx.word_size, isa: init_ctx.isa })
    }
}

impl DiffBackend for QemuBackend {
    fn read_context(&mut self) -> Result<CoreContext, String> {
        let regs = self.gdb.read_regs()?;
        let gprs = self.gpr_names.iter().enumerate()
            .map(|(i, &name)| (name, regs[i]))
            .collect();
        let csrs = self.csr_meta.iter()
            .map(|&(addr, name, mask)| {
                let raw = self.gdb.read_register(csr_addr_to_qemu_regnum(addr))
                    .unwrap_or(0);
                (addr, name, mask, raw)
            })
            .collect();
        let priv_mode = self.gdb.read_register(QEMU_PRIV_REGNUM).unwrap_or(0);
        Ok(CoreContext {
            pc: regs[32], gprs, privilege: priv_mode, csrs,
            word_size: self.word_size, isa: self.isa,
        })
    }

    fn sync_state(&mut self, ctx: &CoreContext) -> Result<(), String> {
        let mut regs: Vec<u64> = ctx.gprs.iter().map(|(_, v)| *v).collect();
        regs.push(ctx.pc);
        self.gdb.write_regs(&regs)?;
        for &(addr, _, _, raw) in &ctx.csrs {
            self.gdb.write_register(csr_addr_to_qemu_regnum(addr), raw)?;
        }
        Ok(())
    }

    fn write_mem(&mut self, addr: usize, data: &[u8]) -> Result<(), String> {
        self.gdb.write_mem(addr, data)
    }

    fn name(&self) -> &str { "qemu" }
}
```

### Phase 5: DiffHarness + diff_contexts — `xdb/src/difftest/mod.rs`

```rust
pub fn diff_contexts(
    dut: &CoreContext, refr: &CoreContext, inst_count: u64,
) -> Option<DiffMismatch> {
    // PC
    if dut.pc != refr.pc {
        return Some(DiffMismatch { inst_count, reg_name: "pc",
            dut_val: dut.pc, ref_val: refr.pc });
    }
    // GPR — validate length, compare by position (skip x0)
    assert_eq!(dut.gprs.len(), refr.gprs.len(), "GPR count mismatch");
    for (i, ((dname, dval), (_, rval))) in dut.gprs.iter().zip(&refr.gprs).enumerate() {
        if i == 0 { continue; }
        if dval != rval {
            return Some(DiffMismatch { inst_count, reg_name: dname,
                dut_val: *dval, ref_val: *rval });
        }
    }
    // Privilege
    if dut.privilege != refr.privilege {
        return Some(DiffMismatch { inst_count, reg_name: "privilege",
            dut_val: dut.privilege, ref_val: refr.privilege });
    }
    // CSRs — match by addr, missing = mismatch
    for &(addr, name, mask, raw) in &dut.csrs {
        let dut_masked = raw & mask;
        match refr.csrs.iter().find(|c| c.0 == addr) {
            Some(&(_, _, _, ref_raw)) => {
                let ref_masked = ref_raw & mask;
                if dut_masked != ref_masked {
                    return Some(DiffMismatch { inst_count, reg_name: name,
                        dut_val: dut_masked, ref_val: ref_masked });
                }
            }
            None => {
                return Some(DiffMismatch { inst_count, reg_name: name,
                    dut_val: dut_masked, ref_val: 0 });
            }
        }
    }
    None
}

impl DiffHarness {
    pub fn new(backend: Box<dyn DiffBackend>) -> Self {
        Self { backend, inst_count: 0 }
    }

    pub fn check_step(&mut self, dut_ctx: &CoreContext, mmio: bool, halted: bool)
        -> Result<Option<DiffMismatch>, String> {
        self.inst_count += 1;
        self.backend.step()?;
        if mmio || halted {
            self.backend.sync_state(dut_ctx)?;
            return Ok(None);
        }
        let ref_ctx = self.backend.read_context()?;
        Ok(diff_contexts(dut_ctx, &ref_ctx, self.inst_count))
    }

    pub fn report_mismatch(m: &DiffMismatch) {
        eprintln!("DIFFTEST MISMATCH at instruction {}:\n  \
                   register: {}\n  DUT: {:#018x}\n  REF: {:#018x}",
                  m.inst_count, m.reg_name, m.dut_val, m.ref_val);
    }

    pub fn inst_count(&self) -> u64 { self.inst_count }
    pub fn backend_name(&self) -> &str { self.backend.name() }
}
```

**`cmd.rs` continue** (R-002 fix — check_step before break):

```rust
pub fn cmd_continue(
    watch_mgr: &mut WatchManager,
    #[cfg(feature = "difftest")] diff: &mut Option<DiffHarness>,
) -> XResult {
    let has_hooks = !watch_mgr.is_empty();
    #[cfg(feature = "difftest")]
    let has_hooks = has_hooks || diff.is_some();

    if !has_hooks {
        return with_xcpu!(run(u64::MAX));
    }

    loop {
        let (ctx, done) = with_xcpu(|cpu| -> XResult<_> {
            cpu.step()?;
            Ok((cpu.context(), cpu.is_terminated()))
        })?;

        // Difftest check BEFORE break — catches halting instruction
        #[cfg(feature = "difftest")]
        if let Some(ref mut h) = diff {
            let mmio = with_xcpu(|cpu| cpu.bus_take_mmio_flag());
            match h.check_step(&ctx, mmio, done) {
                Ok(Some(m)) => { DiffHarness::report_mismatch(&m); return Ok(()); }
                Ok(None) => {}
                Err(e) => { println!("Difftest error: {e}"); *diff = None; return Ok(()); }
            }
        }

        if done { break; }

        if let Some(hit) = check_watchpoints(watch_mgr) {
            println!("{hit}"); return Ok(());
        }
    }
    Ok(())
}
```
