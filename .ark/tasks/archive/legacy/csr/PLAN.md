# CSR Subsystem Implementation Plan

> Phase 2 of [DEV.md](DEV.md) — System Infrastructure
> Full M/S/U privilege support, Rust-idiomatic, embedded in `RVCore`

## Architecture Overview

```
RVCore
├── gpr: [Word; 32]                    (existing)
├── pc / npc: VirtAddr                  (existing)
├── csr: CsrFile                        (new)
├── privilege: PrivilegeMode            (new)
└── pending_trap: Option<PendingTrap>   (new)

CsrFile                                 (dumb storage + descriptor dispatch)
├── regs: [Word; 4096]                  O(1) indexed by 12-bit CSR address
├── read_masked(addr)  → Option<Word>   descriptor-driven, shadow masking
├── write_masked(addr, val) → bool      descriptor-driven, WARL masking
└── get/set(CsrAddr)                    raw access for trap_handler internals

RVCore                                  (policy layer)
├── csr_read(addr)  → ()               privilege check + dynamic rules + trap on fail
├── csr_write(addr, val) → ()          privilege check + side effects + trap on fail
├── raise_trap(cause, tval)            sets pending_trap
└── commit_pending_trap()              trap_entry via npc
```

**Key layering principle**: `CsrFile` handles storage, masks, and shadows. `RVCore` handles
privilege checks, dynamic access rules (TSR/TVM/mcounteren), side effects, and trap generation.
This prevents `CsrFile` from becoming a "half CPU."

## Design Decisions

### 1. Trap signaling: `PendingTrap` struct, fully separated from `XError`

```rust
pub struct PendingTrap {
    pub cause: TrapCause,
    pub tval: Word,
}
```

All architectural traps — ecall, ebreak, illegal instruction, illegal CSR access, page faults —
set `self.pending_trap = Some(PendingTrap { ... })` and return `Ok(())`.

`Err(XError)` is reserved exclusively for:
- Host I/O failures
- Emulator invariant violations
- Implementation bugs

**Why**: Traps are normal RISC-V control flow. `CsrFile::read` returning `Err` for a privilege
violation would mix architectural behavior with emulator faults. Instead, `RVCore::csr_read`
catches the violation and calls `raise_trap`.

### 2. PC commitment: trap writes `npc`, not `pc`

The execute loop has a single commit point: `self.pc = self.npc`.

- Normal instructions: `dispatch()` may update `npc` (branches/jumps)
- Traps: `trap_entry()` sets `self.npc = trap_vector_address`
- The loop then commits: `self.pc = self.npc`

This eliminates the timing conflict where `trap_entry()` writes `pc` and then `execute()`
overwrites it with `npc`.

### 3. Flat 4096-entry array for CSR storage

`[Word; 4096]` indexed directly by 12-bit CSR address. Shadow registers (sstatus, sip, sie)
redirect to the corresponding M-mode register slot with masking — they never occupy their own slot.

### 4. CSR addresses as enum — generated by `csr_table!` macro

The `CsrAddr` enum and the `CSR_DESCS` descriptor table are both generated from a single
`csr_table!` invocation — impossible for them to drift apart. The macro also generates a `match`
dispatch function for O(1) lookup (no linear scan).

### 5. MStatus via `bitflags` crate

```rust
use bitflags::bitflags;

bitflags! {
    #[derive(Debug, Clone, Copy, PartialEq, Eq)]
    pub struct MStatus: Word {
        const SIE   = 1 << 1;
        const MIE   = 1 << 3;
        const SPIE  = 1 << 5;
        const MPIE  = 1 << 7;
        const SPP   = 1 << 8;
        const MPP   = 0b11 << 11;
        const FS    = 0b11 << 13;
        const XS    = 0b11 << 15;
        const MPRV  = 1 << 17;
        const SUM   = 1 << 18;
        const MXR   = 1 << 19;
        const TVM   = 1 << 20;
        const TW    = 1 << 21;
        const TSR   = 1 << 22;
        #[cfg(isa64)]
        const SD    = 1 << 63;
        #[cfg(isa32)]
        const SD    = 1 << 31;

        // Composite masks
        const SSTATUS = Self::SIE.bits() | Self::SPIE.bits() | Self::SPP.bits()
                      | Self::FS.bits()  | Self::XS.bits()
                      | Self::SUM.bits() | Self::MXR.bits() | Self::SD.bits();
        const WRITABLE = Self::SIE.bits() | Self::MIE.bits()
                       | Self::SPIE.bits() | Self::MPIE.bits()
                       | Self::SPP.bits() | Self::MPP.bits()
                       | Self::MPRV.bits() | Self::SUM.bits() | Self::MXR.bits()
                       | Self::TVM.bits() | Self::TW.bits() | Self::TSR.bits()
                       | Self::FS.bits();
    }
}

impl MStatus {
    pub fn mpp(self) -> PrivilegeMode {
        PrivilegeMode::from_bits((self.bits() >> 11) & 0x3)
    }
    pub fn with_mpp(self, mode: PrivilegeMode) -> Self {
        Self::from_bits_truncate(
            (self.bits() & !Self::MPP.bits()) | ((mode as Word) << 11)
        )
    }
    pub fn spp(self) -> PrivilegeMode {
        PrivilegeMode::from_bits((self.bits() >> 8) & 0x1)
    }
    pub fn with_spp(self, mode: PrivilegeMode) -> Self {
        Self::from_bits_truncate(
            (self.bits() & !Self::SPP.bits()) | (((mode as Word) & 1) << 8)
        )
    }
    pub fn sstatus_view(self) -> Word { self.bits() & Self::SSTATUS.bits() }
    pub fn with_sstatus(self, val: Word) -> Self {
        let mask = Self::SSTATUS.bits();
        Self::from_bits_truncate((self.bits() & !mask) | (val & mask))
    }
}
```

### 7. Privilege checking and side effects in RVCore, not CsrFile

**Layering**:
- `CsrFile` only knows about masks and storage
- `RVCore` handles privilege, access rules, side effects, and trap generation
- All violations → `raise_trap(IllegalInstruction)`, never `Err`

**Responsibility boundaries** (important for avoiding confusion as the system grows):
- `AccessRule` in `CsrDesc`: **only** governs whether a CSR *access* is legal at the current privilege.
- `write_side_effects` in `RVCore`: **only** governs CPU-level consequences *after* a legal write completes.
  These are fundamentally different concerns and must not be mixed.
- `TSR` belongs to the `sret` **instruction** path, not CSR access — it stays in `sret()`.
- `TVM` currently affects `satp` CSR access; in future it will also block `SFENCE.VMA` instruction.
  The `BlockedByMstatus` rule covers the CSR access part; the instruction part will be handled
  separately in the instruction dispatcher.

**AccessRule enum** — declarative per-CSR dynamic access rules embedded in `CsrDesc`:

```rust
#[derive(Clone, Copy)]
enum AccessRule {
    /// Standard: privilege level from addr bits [9:8] only
    Standard,
    /// Blocked when a specific mstatus flag is set and privilege == Supervisor
    BlockedByMstatus(MStatus),
    /// Gated by mcounteren / scounteren
    CounterGated,
}
```

**Descriptor with access rule**:

```rust
#[derive(Clone, Copy)]
struct CsrDesc {
    wmask:     Word,
    storage:   u16,
    view_mask: Word,
    access:    AccessRule,
}
```

**Counter bit helper** — encapsulates CSR encoding layout knowledge:

```rust
/// Extract the counteren bit index for a counter CSR address.
/// Counter CSRs 0xC00..=0xC1F map to bits 0..31 of mcounteren/scounteren.
fn counteren_bit(addr: u16) -> u32 {
    (addr & 0x1F) as u32
}
```

**Core pipeline: existence check happens once, descriptor flows through**:

```rust
impl RVCore {
    pub fn csr_read(&mut self, addr: u16) -> Option<Word> {
        // 1. Existence — single lookup, unknown CSR = immediate trap
        let Some(desc) = find_desc(addr) else {
            self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
            return None;
        };
        // 2. Permission — privilege + dynamic rules, desc passed in (no re-lookup)
        if !self.check_csr_access(addr, &desc) { return None; }
        // 3. Read — guaranteed to succeed (desc exists)
        let val = self.csr.read_with_desc(&desc);
        self.read_side_effects(addr);
        Some(val)
    }

    pub fn csr_write(&mut self, addr: u16, val: Word) -> bool {
        // 1. Existence
        let Some(desc) = find_desc(addr) else {
            self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
            return false;
        };
        // 2. Read-only check
        if Self::is_read_only(addr) {
            self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
            return false;
        }
        // 3. Permission
        if !self.check_csr_access(addr, &desc) { return false; }
        // 4. Write — guaranteed to succeed (desc exists)
        self.csr.write_with_desc(&desc, val);
        self.write_side_effects(addr);
        true
    }

    /// Dynamic access rules — desc is already resolved, no re-lookup needed.
    fn check_csr_access(&mut self, addr: u16, desc: &CsrDesc) -> bool {
        // Basic privilege check (always applies)
        let required = PrivilegeMode::from_bits(((addr >> 8) & 0x3) as Word);
        if self.privilege < required {
            self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
            return false;
        }
        // Per-CSR dynamic rule from descriptor
        match desc.access {
            AccessRule::Standard => true,
            AccessRule::BlockedByMstatus(flag) => {
                let ms = MStatus::from_bits_truncate(self.csr.get(CsrAddr::mstatus));
                if ms.contains(flag) && self.privilege == PrivilegeMode::Supervisor {
                    self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
                    false
                } else {
                    true
                }
            }
            AccessRule::CounterGated => {
                let bit = counteren_bit(addr);
                match self.privilege {
                    PrivilegeMode::Machine => true,
                    PrivilegeMode::Supervisor => {
                        if (self.csr.get(CsrAddr::mcounteren) >> bit) & 1 == 0 {
                            self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
                            false
                        } else { true }
                    }
                    PrivilegeMode::User => {
                        let m_ok = (self.csr.get(CsrAddr::mcounteren) >> bit) & 1 == 1;
                        let s_ok = (self.csr.get(CsrAddr::scounteren) >> bit) & 1 == 1;
                        if !(m_ok && s_ok) {
                            self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
                            false
                        } else { true }
                    }
                }
            }
        }
    }

    /// Side effects for CSR writes. CPU-level policy, NOT CSR metadata.
    /// This is the sole side-effect entry point (no fn-pointer hooks in CsrDesc).
    ///
    /// Current side-effect map:
    ///   satp     (0x180) — Phase 3: flush TLB / mark address translation dirty
    ///   mstatus  (0x300) — Phase 3: MPRV/SUM/MXR affect memory access translation
    ///   mie      (0x304) — Phase D: may need to re-evaluate pending interrupts
    ///   mip      (0x344) — Phase D: writing MSIP/SSIP may trigger immediate interrupt
    ///   mideleg  (0x303) — Phase D: changes interrupt routing
    ///
    /// CSRs with NO side effects (just storage): mepc, mcause, mtval, mscratch,
    /// sepc, scause, stval, sscratch, medeleg, mtvec, stvec, mcounteren, scounteren
    ///
    /// NOT in this function (different concern):
    ///   TSR — belongs to sret instruction path, not CSR access
    ///   TVM on SFENCE.VMA — belongs to instruction dispatcher (future)
    fn write_side_effects(&mut self, addr: u16) {
        match addr {
            0x180 => { /* satp: placeholder for Phase 3 TLB flush */ }
            _     => {}
        }
    }
}
```

**CsrFile gains desc-aware methods** (no redundant lookup):

```rust
impl CsrFile {
    /// Read using a pre-resolved descriptor. Cannot fail (desc is known valid).
    pub fn read_with_desc(&self, desc: CsrDesc) -> Word {
        self.regs[desc.storage as usize] & desc.view_mask
    }

    /// Write using a pre-resolved descriptor. Cannot fail (desc is known valid).
    pub fn write_with_desc(&mut self, desc: CsrDesc, val: Word) {
        let slot = &mut self.regs[desc.storage as usize];
        let mask = desc.view_mask & desc.wmask;
        *slot = (*slot & !mask) | (val & mask);
    }

    /// Legacy: lookup + read in one call. Returns None for unknown CSR.
    pub fn read_masked(&self, addr: u16) -> Option<Word> {
        Some(self.read_with_desc(find_desc(addr)?))

    /// Legacy: lookup + write in one call. Returns false for unknown CSR.
    pub fn write_masked(&mut self, addr: u16, val: Word) -> bool {
        let Some(desc) = find_desc(addr) else { return false };
        self.write_with_desc(desc, val);
        true
    }
}
```

## The `csr_table!` Macro

```rust
macro_rules! csr_table {
    ( $( $name:ident = $addr:expr, $($spec:tt)+ );* $(;)? ) => {
        #[derive(Clone, Copy, Debug, PartialEq, Eq, IntoPrimitive, TryFromPrimitive)]
        #[repr(u16)]
        #[allow(non_camel_case_types)]
        pub enum CsrAddr {
            $( $name = $addr ),*
        }

        /// O(1) descriptor lookup via match (compiler → jump table)
        fn find_desc(addr: u16) -> Option<CsrDesc> {
            match addr {
                $( $addr => Some(csr_table!(@desc $addr, $($spec)+)), )*
                _ => None,
            }
        }
    };

    // Normal RW register (default: Standard access)
    (@desc $addr:expr, RW($wmask:expr)) => {
        CsrDesc { wmask: $wmask, storage: $addr, view_mask: !0, access: AccessRule::Standard }
    };
    // Shadow register (default: Standard access)
    (@desc $addr:expr, RW($wmask:expr) => $alias:ident ($vmask:expr)) => {
        CsrDesc { wmask: $wmask, storage: CsrAddr::$alias as u16, view_mask: $vmask, access: AccessRule::Standard }
    };
    // Read-only register (default: Standard access)
    (@desc $addr:expr, RO) => {
        CsrDesc { wmask: 0, storage: $addr, view_mask: !0, access: AccessRule::Standard }
    };

    // ---- Variants with access rule annotations ----

    // RW + blocked_by(flag)
    (@desc $addr:expr, RW($wmask:expr), blocked_by($flag:expr)) => {
        CsrDesc { wmask: $wmask, storage: $addr, view_mask: !0, access: AccessRule::BlockedByMstatus(MStatus::$flag) }
    };
    // RO + counter_gated
    (@desc $addr:expr, RO, counter_gated) => {
        CsrDesc { wmask: 0, storage: $addr, view_mask: !0, access: AccessRule::CounterGated }
    };
    // RW + counter_gated (for future writable counters)
    (@desc $addr:expr, RW($wmask:expr), counter_gated) => {
        CsrDesc { wmask: $wmask, storage: $addr, view_mask: !0, access: AccessRule::CounterGated }
    };
}
```

### CSR Registration

```rust
// Interrupt bit masks
const MIE_WMASK: Word = (1 << 1) | (1 << 3) | (1 << 5) | (1 << 7) | (1 << 9) | (1 << 11);
const MIP_WMASK: Word = (1 << 1) | (1 << 3);  // Only SSIP, MSIP software-writable
const SIE_MASK: Word  = (1 << 1) | (1 << 5) | (1 << 9);  // SSI, STI, SEI
const SIP_MASK: Word  = (1 << 1);  // Only SSIP writable from S-mode

csr_table! {
    // ---- M-mode Trap Setup ----
    mstatus    = 0x300, RW(MStatus::WRITABLE.bits());
    misa       = 0x301, RO;
    medeleg    = 0x302, RW(0xB3FF);          // delegatable exception bits
    mideleg    = 0x303, RW(0x222);           // S-mode interrupt bits (SSI|STI|SEI)
    mie        = 0x304, RW(MIE_WMASK);
    mtvec      = 0x305, RW(!(0x3 as Word));  // force direct mode (mode bits [1:0] = 0)
    mcounteren = 0x306, RW(0x7);             // CY, TM, IR bits

    // ---- M-mode Trap Handling ----
    mscratch   = 0x340, RW(!0);
    mepc       = 0x341, RW(!0);
    mcause     = 0x342, RW(!0);
    mtval      = 0x343, RW(!0);
    mip        = 0x344, RW(MIP_WMASK);

    // ---- S-mode Shadows ----
    sstatus    = 0x100, RW(MStatus::SSTATUS.bits()) => mstatus(MStatus::SSTATUS.bits());
    sie        = 0x104, RW(SIE_MASK)                => mie(SIE_MASK);
    sip        = 0x144, RW(SIP_MASK)                => mip(SIP_MASK);

    // ---- S-mode Own Registers ----
    stvec      = 0x105, RW(!(0x3 as Word));  // force direct mode (mode bits [1:0] = 0)
    scounteren = 0x106, RW(0x7);
    sscratch   = 0x140, RW(!0);
    sepc       = 0x141, RW(!0);
    scause     = 0x142, RW(!0);
    stval      = 0x143, RW(!0);
    satp       = 0x180, RW(!0), blocked_by(TVM);  // TVM blocks S-mode access

    // ---- Machine Information (read-only) ----
    mvendorid  = 0xF11, RO;
    marchid    = 0xF12, RO;
    mhartid    = 0xF14, RO;

    // ---- Counters (read-only from U/S, gated by mcounteren/scounteren) ----
    cycle      = 0xC00, RO, counter_gated;
    instret    = 0xC02, RO, counter_gated;
}
```

## Implementation Phases

### Phase A: Trap Channel + PC Fix

**Goal**: Establish `PendingTrap` and fix the pc/npc commit model.

- Add `PendingTrap` struct, `TrapCause`, `Exception`, `Interrupt` enums
- Add `PrivilegeMode` enum
- Add `pending_trap: Option<PendingTrap>` and `privilege: PrivilegeMode` to `RVCore`
- `ecall` → sets `pending_trap`, returns `Ok(())`
- `ebreak` → sets `pending_trap(Breakpoint)`, returns `Ok(())`
- `execute()` loop: check `pending_trap` after dispatch, trap writes `npc` not `pc`
- Batch runner: catch `Breakpoint` trap and treat as termination

**Tests**: ecall/ebreak set pending_trap correctly, pc/npc commit order.

### Phase B: Minimal CsrFile

**Goal**: Storage + descriptors + MStatus bitflags.

- `CsrFile` with `[Word; 4096]`, `get`/`set`, `read_masked`/`write_masked`
- `csr_table!` macro generating `CsrAddr` + `find_desc` match
- `MStatus` via `bitflags!` with all field accessors
- Register full CSR table (mstatus, medeleg, mideleg, mie, mip, mtvec, sstatus, etc.)
- Explicit wmasks: `MIP_WMASK`, `MIE_WMASK`, `MEDELEG_WMASK`
- `mcounteren` and `scounteren` included from the start

**Tests**: read/write each CSR, shadow consistency (sstatus↔mstatus), WARL masking, unknown CSR → None.

### Phase C: Zicsr + Trap Return + Policy Layer

**Goal**: CSR instructions work with full privilege checking.

- `RVCore::csr_read`/`csr_write` — privilege check, dynamic rules (TSR/TVM/counteren), raise_trap
- Implement csrrw/csrrs/csrrc/csrrwi/csrrsi/csrrci using `self.csr_read`/`csr_write`
- Implement `trap_entry` (writes `npc`), `do_mret`, `do_sret`
- `mret`: privilege check + clear MPRV if returning below M
- `sret`: TSR check + privilege check

**Tests**: CSR instruction semantics (rd=x0, rs1=x0), privilege violations → trap, TSR blocking sret,
RO CSR write → trap, full trap roundtrip (ecall→trap_entry→mret).

### Phase D: Interrupt Framework

**Goal**: Basic interrupt priority and delegation.

- `check_pending_interrupt()` with priority scanning
- Delegation via `mideleg`
- Global enable gating via `MIE`/`SIE`
- Interrupt sampling in `step()` **before** `fetch()`, not inside `execute()`:
  interrupts are asynchronous and should be checked between instructions,
  before the next instruction is fetched. This matches real hardware behavior
  and avoids contaminating the current instruction's pc/npc state.

**Tests**: priority ordering, delegation, MIE=0 blocks M-level, U-mode always takes delegated.

### Phase E: Counters

**Goal**: cycle/instret working with access control.

- `csr.increment_counters()` at end of execute (not for trapped instructions)
- `mcounteren`/`scounteren` enforce access from S/U mode (already in Phase B table)
- `instret` does not increment on trapping instructions

**Tests**: counter increments, access control from S/U mode.

## TrapCause Enums

```rust
#[derive(Clone, Copy, Debug, PartialEq, Eq)]
#[repr(u8)]
pub enum Exception {
    InstructionMisaligned  = 0,
    InstructionAccessFault = 1,
    IllegalInstruction     = 2,
    Breakpoint             = 3,
    LoadMisaligned         = 4,
    LoadAccessFault        = 5,
    StoreMisaligned        = 6,
    StoreAccessFault       = 7,
    EcallFromU             = 8,
    EcallFromS             = 9,
    EcallFromM             = 11,
    InstructionPageFault   = 12,
    LoadPageFault          = 13,
    StorePageFault         = 15,
}

#[derive(Clone, Copy, Debug, PartialEq, Eq)]
#[repr(u8)]
pub enum Interrupt {
    SupervisorSoftware  = 1,
    MachineSoftware     = 3,
    SupervisorTimer     = 5,
    MachineTimer        = 7,
    SupervisorExternal  = 9,
    MachineExternal     = 11,
}

#[derive(Clone, Copy, Debug, PartialEq, Eq)]
pub enum TrapCause {
    Exception(Exception),
    Interrupt(Interrupt),
}

impl TrapCause {
    pub fn is_interrupt(&self) -> bool { matches!(self, Self::Interrupt(_)) }
    pub fn code(&self) -> Word {
        match self {
            Self::Exception(e) => *e as Word,
            Self::Interrupt(i) => *i as Word,
        }
    }
    pub fn to_mcause(&self) -> Word {
        match self {
            Self::Exception(_) => self.code(),
            Self::Interrupt(_) => (1 as Word).wrapping_shl(Word::BITS - 1) | self.code(),
        }
    }
}
```

## Trap Entry & Exit

```rust
impl RVCore {
    pub fn raise_trap(&mut self, cause: TrapCause, tval: Word) {
        self.pending_trap = Some(PendingTrap { cause, tval });
    }

    /// Called from execute loop when pending_trap is Some.
    /// Writes npc (not pc) — the loop commits pc = npc afterward.
    pub fn commit_pending_trap(&mut self) {
        let trap = self.pending_trap.take().unwrap();
        let delegated = self.is_delegated(&trap.cause);

        if delegated {
            let mut ms = MStatus::from_bits_truncate(self.csr.get(CsrAddr::mstatus));
            self.csr.set(CsrAddr::sepc, self.pc.as_usize() as Word);
            self.csr.set(CsrAddr::scause, trap.cause.to_mcause());
            self.csr.set(CsrAddr::stval, trap.tval);
            ms = ms.with_spp(self.privilege);
            ms.set(MStatus::SPIE, ms.contains(MStatus::SIE));
            ms.remove(MStatus::SIE);
            self.csr.set(CsrAddr::mstatus, ms.bits());
            self.privilege = PrivilegeMode::Supervisor;
            self.npc = VirtAddr::from(self.csr.get(CsrAddr::stvec) as usize & !0x3);
        } else {
            let mut ms = MStatus::from_bits_truncate(self.csr.get(CsrAddr::mstatus));
            self.csr.set(CsrAddr::mepc, self.pc.as_usize() as Word);
            self.csr.set(CsrAddr::mcause, trap.cause.to_mcause());
            self.csr.set(CsrAddr::mtval, trap.tval);
            ms = ms.with_mpp(self.privilege);
            ms.set(MStatus::MPIE, ms.contains(MStatus::MIE));
            ms.remove(MStatus::MIE);
            self.csr.set(CsrAddr::mstatus, ms.bits());
            self.privilege = PrivilegeMode::Machine;
            self.npc = VirtAddr::from(self.csr.get(CsrAddr::mtvec) as usize & !0x3);
        }
    }

    pub fn do_mret(&mut self) {
        let mut ms = MStatus::from_bits_truncate(self.csr.get(CsrAddr::mstatus));
        let mpp = ms.mpp();
        ms.set(MStatus::MIE, ms.contains(MStatus::MPIE));
        ms.insert(MStatus::MPIE);
        ms = ms.with_mpp(PrivilegeMode::User);
        if mpp != PrivilegeMode::Machine { ms.remove(MStatus::MPRV); }
        self.csr.set(CsrAddr::mstatus, ms.bits());
        self.privilege = mpp;
        self.npc = VirtAddr::from(self.csr.get(CsrAddr::mepc) as usize);
    }

    pub fn do_sret(&mut self) {
        let mut ms = MStatus::from_bits_truncate(self.csr.get(CsrAddr::mstatus));
        let spp = ms.spp();
        ms.set(MStatus::SIE, ms.contains(MStatus::SPIE));
        ms.insert(MStatus::SPIE);
        ms = ms.with_spp(PrivilegeMode::User);
        ms.remove(MStatus::MPRV);  // sret always returns to S/U (< M), must clear MPRV
        self.csr.set(CsrAddr::mstatus, ms.bits());
        self.privilege = spp;
        self.npc = VirtAddr::from(self.csr.get(CsrAddr::sepc) as usize);
    }
}
```

### Execute Loop

```rust
fn execute(&mut self, inst: DecodedInst) -> XResult {
    let is_compressed = matches!(&inst, DecodedInst::C { .. });
    self.npc = self.pc.wrapping_add(if is_compressed { 2 } else { 4 });
    self.dispatch(inst)?;

    let trapped = self.pending_trap.is_some();
    if trapped {
        self.commit_pending_trap();  // takes pending_trap, clears it
    }

    self.pc = self.npc;
    if !trapped {
        self.csr.increment_counters();
    }
    Ok(())
}
```

### Privileged Instructions

```rust
pub(super) fn ecall(&mut self, _rd: RVReg, _rs1: RVReg, _imm: SWord) -> XResult {
    self.raise_trap(TrapCause::Exception(match self.privilege {
        PrivilegeMode::User       => Exception::EcallFromU,
        PrivilegeMode::Supervisor => Exception::EcallFromS,
        PrivilegeMode::Machine    => Exception::EcallFromM,
    }), 0);
    Ok(())
}

pub(super) fn ebreak(&mut self, _rd: RVReg, _rs1: RVReg, _imm: SWord) -> XResult {
    self.raise_trap(TrapCause::Exception(Exception::Breakpoint), self.pc.as_usize() as Word);
    Ok(())
}

pub(super) fn mret(&mut self, _rd: RVReg, _rs1: RVReg, _rs2: RVReg) -> XResult {
    if self.privilege < PrivilegeMode::Machine {
        self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
        return Ok(());
    }
    self.do_mret();
    Ok(())
}

pub(super) fn sret(&mut self, _rd: RVReg, _rs1: RVReg, _rs2: RVReg) -> XResult {
    if self.privilege < PrivilegeMode::Supervisor {
        self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
        return Ok(());
    }
    let ms = MStatus::from_bits_truncate(self.csr.get(CsrAddr::mstatus));
    if ms.contains(MStatus::TSR) && self.privilege == PrivilegeMode::Supervisor {
        self.raise_trap(TrapCause::Exception(Exception::IllegalInstruction), 0);
        return Ok(());
    }
    self.do_sret();
    Ok(())
}
```

## Notes

- `RVCore` starts in M-mode after reset, per RISC-V spec
- `satp` stored but SV39/SV32 page walking is Phase 3 (write side-effect hook is a placeholder)
- `mret` clears `MPRV` when returning to privilege < M (per spec)
- Counter CSRs use array slots; on RV32, `cycleh`/`instreth` can be added later
- `tval` carries faulting address for page faults (Phase 3) and PC for ebreak
- Vectored mtvec mode (base + 4×cause) is a stretch goal — currently wmask forces direct mode only
- No `fn` pointer hooks in `CsrDesc` — side effects live in `RVCore::write_side_effects()` match
